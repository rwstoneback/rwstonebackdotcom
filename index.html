Stuff I've done

SensorCloud

I worked with a team in redesigning and developing the front-end of sensorcloud.com. Utilizing bootstrap, we were able to make the site mobile and tablet friendly.

Unity Data Visualization

Using the Unity3D Game Engine, I read in live data from wireless sensors to create 3D interactive demos to better visualize the information. These are being used at trade shows to help demonstrate the cability and use cases of the hardware.

MSCL

MSCL is a library that allows others to interface with MicroStrain's hardware at a high-level. By utilizing SWIG, MSCL is available for C++, Python, C#, VB, and even LabView. Both external and internal users can now write software using the same simple, tested library.

Nyctophobia

During my senior year at Champlain College, I worked with a team to develop the PC game, Nyctophobia. This was developed using the Unreal Development Kit. Nyctophobia is a horror puzzle/platformer game in which the player must switch between the human world and the "demon" world in order to escape a haunted house. Watch the Trailer.
